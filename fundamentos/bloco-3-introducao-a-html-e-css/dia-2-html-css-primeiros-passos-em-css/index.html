Requisitos

Seu Portfólio Web deve ser estilizado usando as informações a seguir:

  1. Defina fontes diferentes para o seu nome e para a descrição que você criou;

  2. Defina uma cor base de background do seu Portfólio Web ;

  3. Altere o estilo das tags que você usou para destacar algumas informações,
    como sua nacionalidade e a cidade/estado onde mora;

  4. Coloque tamanhos diferentes para os elementos da lista de habilidades que você criou.
    Lembre-se de usar classes para cada um dos elementos;
